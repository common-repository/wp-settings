== Description ==

Plugin for displaying important wordpress environment details such as the general wordpress settings,php settings(such as version),server and mysql details.You can Generate Database Backup Script
which could be used for restoring database.There are many settings which should be configured properly and WP Settings plugin will help you see all the useful settings from a single location.You 
can see the information about the general wordpress settings,php settings,mysql,server and installed plugins.


Some of the options in this plugin:

* WordPress settings such as version and url.

* MySQL details such as version and database.

* Generate Database Backup Script

* Server details such as Apache version.

* PHP settings.

* Plugin Details.

* Using this plugin you can generate the script for creating database tables.


== Installation ==
1. Download via Plugins - Add New.You can also download wp-settings.zip and upload it to the `/wp-content/plugins` directory
2. Activate the `WP Settings` plugin.
3. View different details related to Themes,Php,MySQL in menu `WP Settings`
4. Generate script for creating database backup
